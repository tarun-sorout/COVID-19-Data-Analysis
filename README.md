# 🦠 COVID-19 Data Analysis

## 📌 Overview

The COVID-19 pandemic had a significant impact on countries around the world. This project analyzes global COVID-19 data to understand the spread of the virus, recovery trends, death rates, and country-wise impacts.

Using Python and data visualization techniques, the project explores patterns in confirmed, recovered, active, and death cases across different countries and regions. The analysis helps identify trends and provides insights into the progression of the pandemic over time.

---

## 🎯 Objectives

- Analyze global COVID-19 statistics.
- Identify countries with the highest number of cases.
- Study recovery and mortality trends.
- Examine the growth of COVID-19 over time.
- Visualize data using charts and graphs.
- Generate meaningful insights from pandemic data.

---

## 📂 Dataset Information

The dataset contains COVID-19 records from multiple countries and regions.

### Features

| Column | Description |
|----------|-------------|
| Province/State | State or Province Name |
| Country/Region | Country Name |
| Lat | Latitude |
| Long | Longitude |
| Date | Reporting Date |
| Confirmed | Total Confirmed Cases |
| Deaths | Total Death Cases |
| Recovered | Total Recovered Cases |
| WHO Region | WHO Classification Region |

### Derived Feature

| Column | Description |
|----------|-------------|
| Active | Confirmed − Deaths − Recovered |

## 🔍 Key Insights

- Confirmed cases increased rapidly during the pandemic period.
- Recovery numbers grew steadily over time.
- Countries with higher confirmed cases generally reported higher death counts.
- A small number of countries contributed a significant share of global cases.
- Recovery rates improved as healthcare systems adapted to the pandemic.
- The distribution of cases varied considerably across WHO regions.
